# UIDAI Aadhaar Enrollment Analysis – 2025

This project presents an Exploratory Data Analysis (EDA) of Aadhaar enrollment data released by UIDAI as part of the UIDAI Data Hackathon 2026.

The analysis focuses on understanding Aadhaar enrollment trends across regions, districts, and age groups in India for the year 2025 using Python-based data analysis and visualization techniques.

## Project Objective
The objective of this project is to analyze Aadhaar enrollment patterns and extract insights that can support data-driven decision-making for enrollment planning, infrastructure allocation, and demographic analysis.

## Dataset Information
* Dataset: Aadhaar Enrollment Dataset
* Source: UIDAI (Unique Identification Authority of India)
* Data Type: Aggregated and anonymized enrollment records
* Time Period: March 2025 – December 2025
* Granularity: District-level data

The dataset was provided in multiple CSV files and merged for analysis.

## Analysis Performed
* Monthly Aadhaar enrollment trend analysis
* State-wise enrollment comparison
* Age-group enrollment distribution
* District-level enrollment hotspot identification
* Monthly enrollment volume visualization
* District hotspot density analysis

## Key Insights
* Aadhaar enrollments showed noticeable monthly variation during 2025.
* The 0–5 age group contributed the highest share of enrollments.
* A limited number of states accounted for a major portion of total enrollments.
* Certain districts emerged as enrollment hotspots, indicating concentrated enrollment activity.
